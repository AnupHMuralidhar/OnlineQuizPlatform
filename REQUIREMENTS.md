# Project Requirements

This document lists all system-level and software dependencies required
to build and run the Online Quiz & Assessment Platform.

---

## 1. System Requirements

- Operating System: Windows 10 or later
- Architecture: 64-bit
- Internet connection (for initial dependency installation)

---

## 2. Required Software (Global)

The following must be installed on the system:

### 2.1 Node.js
- Version: Node.js LTS (v18 or later)
- Includes: npm (Node Package Manager)

Used for:
- Backend server (Express.js)
- Frontend application (React)

---

### 2.2 Code Editor
- Visual Studio Code (recommended)

Used for:
- Writing and managing source code
- Running integrated terminal commands

---

### 2.3 Web Browser
- Google Chrome / Firefox / Edge

Used for:
- Accessing the frontend UI
- Testing application behavior

---
