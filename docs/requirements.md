# Project Requirements

## Project Title
Online Quiz & Assessment Platform

## Objective
To design and implement a simple online quiz and assessment platform that allows users to create quizzes, attempt them, receive automatic scores, and view past quiz attempts.

The primary goal of this project is to fully satisfy the minimum functional requirements with a working application.

---

## Functional Requirements

1. Users can register and log in using a simple username and password.
2. Users can create quizzes.
3. Each quiz contains multiple questions.
4. Questions support multiple-choice answers.
5. Users can attempt available quizzes.
6. Quiz scores are calculated automatically.
7. Users can view their past quiz attempts and scores.

---

## Non-Functional Requirements

- The application must run locally on Windows.
- The system must be built using free and open-source technologies.
- The user interface should be clean, simple, and functional.
- The system should be easy to understand and extend.

---

## Scope Limitations (v1)

- Only one general domain of questions is used.
- No difficulty levels are implemented.
- No adaptive or personalized quiz logic.
- No analytics or reporting dashboards.
- No machine learning features.

These features are considered future enhancements and are not part of version 1.
